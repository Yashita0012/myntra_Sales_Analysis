# 🛍️ Myntra Fashion Sales & Customer Insights Dashboard

An end-to-end Data Analytics project that analyzes over **526,000+ Myntra fashion products** to uncover customer preferences, pricing strategies, discount trends, brand performance, and category insights.

The project combines **Python (Pandas, Matplotlib, Seaborn)** for data cleaning and exploratory data analysis (EDA) with **Power BI** for interactive dashboard visualization. :contentReference[oaicite:0]{index=0}

---

## 📌 Project Overview

The objective of this project is to analyze Myntra's fashion product dataset and answer business questions such as:

- Which brands dominate the platform?
- Which product categories generate the highest listings?
- How are discounts distributed across brands?
- What is the relationship between ratings and customer reviews?
- Which gender segment contributes the most products?
- Which sizes are most commonly available?
- Which categories should receive more marketing focus?

## 🔄 Data Cleaning

The following preprocessing steps were performed:

- Removed unnecessary columns
- Handled missing values
- Imputed missing Ratings, Reviews, and Discount Price using mean values
- Dropped Discount Offer column due to excessive missing values
- Separated categorical and numerical features
- Verified data types
- Performed exploratory data analysis (EDA) :contentReference[oaicite:1]{index=1}

## 💡 Business Insights

Some important insights generated from the analysis:

- Indian Wear and Western Wear dominate the product catalog.
- Pothys, Roadster, and KALINI are among the largest brands.
- Women account for the majority of listed products.
- Average product rating is approximately **4.09**, indicating generally positive customer feedback.
- Average discount offered is approximately **42%**.
- Products with higher ratings tend to receive more customer reviews.
- Standard sizes (S, M, L, XL) are the most commonly available across products.
